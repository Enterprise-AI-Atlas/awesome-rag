# AGENTS.md — Awesome RAG

## Purpose

This repository is a curated registry of retrieval-augmented generation (RAG) frameworks, tools, and reference implementations.

## Scope

Cover resources that help build production RAG systems:

- RAG frameworks and orchestration libraries.
- Embedding models and retrieval engines.
- Chunking, parsing, and document-processing tools.
- Evaluation and observability tools for RAG pipelines.
- Vector databases and hybrid search systems.

## Style

- Follow the same format as the other `awesome-*` repositories in the Enterprise AI Atlas organization.
- Label official/vendor resources as `Official` and third-party resources as `Community`.
- Prefer official documentation and links.

## Maintenance

- Run `./scripts/validate-links.sh` before finishing any edit.
- Place entries under the RAG pipeline stage or tool category they primarily target.

## Important reminders

- Do not provide legal advice.
- Verify that links resolve and are current.
